# AGENTS

- Unless otherwise specificed, always verify that your code changes pass existing tests and builds properly with no typescript type errors
- When writing a class, always include a comment above the class describing its basic function. If editing a class, ensure that the existing comment is still accurate, and if not update it to ensure accuracy
- If writing a function, always include a comment above the function including the following: a brief description of the purpose of the function, @param tags to describe its parameters, and an @returns describing the return type. If editing a function, ensure that the existing comment is still accurate, and if not update it to ensure accuracy
- Always ensure code is properly decomposed, and only introduce dependencies when necessary
